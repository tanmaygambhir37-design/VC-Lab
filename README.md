# VC Lab: From Startup Screening to Investment Decision
Interactive venture capital decision engine: startup screening, valuation, cap tables, dilution and return analysis.
## Objective
Build a hands-on venture capital simulation platform to learn how investors evaluate startups, structure deals, manage dilution, and generate returns.

This project combines:
- Startup screening
- Valuation methodologies
- Cap table modeling
- Portfolio return analysis
- Investment committee decision-making

---

## Scope

### Phase 1 — Startup Scorecard
Evaluate startups using weighted factors:
- Team
- Market
- Product
- Traction
- Moat

### Phase 2 — Valuation Engine
Build:
- Comparable valuation
- VC Method
- Scorecard valuation

### Phase 3 — Cap Table Simulator
Model:
- Ownership
- Dilution
- Follow-on rounds

### Phase 4 — Return Engine
Analyze:
- MOIC
- IRR
- Exit scenarios

### Phase 5 — Investment Committee Memo
Generate investment recommendations.
